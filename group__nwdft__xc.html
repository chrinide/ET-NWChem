subroutine setacmat ( double precision, dimension(nq,3,ipol)  delrho,
double precision, dimension(nq,ipol)  Amat,
double precision, dimension(nq,3,ipol)  Cmat,
double precision, dimension(nq)  qwght,

logical  kske,
double precision, dimension(nq,ipol)  Mmat

Do clever stuff to the partial derivatives.

The density functionals return plain partial derivatives of the of the functionals in addition to the energy. This routine modifies those quantities to get them in a form that is more "convenient" in the construction of various matrices.

In all cases this modification includes multiplying the partial derivatives for each grid point with the corresponding quadrature weight.

If xform_Cmat is set to .TRUE. then the 1st derivatives with respect to the norm of the electron density gradient are replaced in the closed shell case with

\begin{eqnarray*} Cmat(i,1,1) &=& \frac{\partial\rho(r)}{\partial x} \left( \frac{\partial f}{\partial\gamma^{\alpha\alpha}} + \frac{1}{2}\frac{\partial f}{\partial\gamma^{\alpha\beta}} \right) w_i\\\ Cmat(i,2,1) &=& \frac{\partial\rho(r)}{\partial y} \left( \frac{\partial f}{\partial\gamma^{\alpha\alpha}} + \frac{1}{2}\frac{\partial f}{\partial\gamma^{\alpha\beta}} \right) w_i\\\ Cmat(i,3,1) &=& \frac{\partial\rho(r)}{\partial z} \left( \frac{\partial f}{\partial\gamma^{\alpha\alpha}} + \frac{1}{2}\frac{\partial f}{\partial\gamma^{\alpha\beta}} \right) w_i\\\ \end{eqnarray*}

and in the open shell case with

\begin{eqnarray*} Cmat(i,1,1) &=& \left(2\frac{\partial\rho^\alpha(r)}{\partial x} \frac{\partial f}{\partial\gamma^{\alpha\alpha}} + \frac{\partial\rho^\beta(r)}{\partial x} \frac{\partial f}{\partial\gamma^{\alpha\beta}}\right)w_i \\\ Cmat(i,2,1) &=& \left(2\frac{\partial\rho^\alpha(r)}{\partial y} \frac{\partial f}{\partial\gamma^{\alpha\alpha}} + \frac{\partial\rho^\beta(r)}{\partial y} \frac{\partial f}{\partial\gamma^{\alpha\beta}}\right)w_i \\\ Cmat(i,3,1) &=& \left(2\frac{\partial\rho^\alpha(r)}{\partial z} \frac{\partial f}{\partial\gamma^{\alpha\alpha}} + \frac{\partial\rho^\beta(r)}{\partial z} \frac{\partial f}{\partial\gamma^{\alpha\beta}}\right)w_i \\\ Cmat(i,1,2) &=& \left(2\frac{\partial\rho^\beta(r)}{\partial x} \frac{\partial f}{\partial\gamma^{\beta\beta}} + \frac{\partial\rho^\alpha(r)}{\partial x} \frac{\partial f}{\partial\gamma^{\alpha\beta}}\right)w_i \\\ Cmat(i,2,2) &=& \left(2\frac{\partial\rho^\beta(r)}{\partial y} \frac{\partial f}{\partial\gamma^{\beta\beta}} + \frac{\partial\rho^\alpha(r)}{\partial y} \frac{\partial f}{\partial\gamma^{\alpha\beta}}\right)w_i \\\ Cmat(i,3,2) &=& \left(2\frac{\partial\rho^\beta(r)}{\partial z} \frac{\partial f}{\partial\gamma^{\beta\beta}} + \frac{\partial\rho^\alpha(r)}{\partial z} \frac{\partial f}{\partial\gamma^{\alpha\beta}}\right)w_i \\\ \end{eqnarray*}

The grid point is indexed with i, the density functional is referred to with \(f\), and \(w\) is the quadrature weight.

Parameters
ipol[Input] The number of spin channels
nq[Input] The number of grid points
grad[Input] If .TRUE. the density functional is a GGA
xform_cmat[Input] If .TRUE. do the elaborate transformation
delrho[Input] The gradient of the density
amat[In/Output] \(\partial f/\parti! al\rho^\sigma\)
cmat[In/Output] \(\partial f/\par! tial\gamma^{\sigma\sigma'}\)
mmat[In/Output] \(\partial f/\parti! al\tau^\sigma\)
kske[Input] If .TRUE. the density functional is a meta-GGA
qwght[Input] The quadrature weights

Evaluate various matrix elements over the functionals.

This routine evaluates a variety of matrix elements that involve the density functional.

Note that there are some inconsistencies with respect to the kinetic energy density. NWChem defines the kinetic energy density as

\begin{eqnarray*} \tau_1 &=& \sum_i \frac{1}{2}\langle\phi_i|\nabla^2|\phi_i\rangle \end{eqnarray*}

In most density functionals the kinetic energy density is defined as

\begin{eqnarray*} \tau_2 &=& \sum_i \langle\phi_i|\nabla^2|\phi_i\rangle \end{eqnarray*}

the reason being that the factor 1/2 can easily be absorpted into the density functional parameters. As a result the density functional routines evaluate

\begin{eqnarray*} e &=& f(\rho,\gamma,\tau_2) \\\ Amat = &=& \frac{\partial f(\rho,\gamma,\tau_2)}{\partial\rho} \\\ Cmat = &=& \frac{\partial f(\rho,\gamma,\tau_2)}{\partial\gamma} \\\ Mmat = &=& \frac{\partial f(\rho,\gamma,\tau_2)}{\partial\tau_2} \\\ \end{eqnarray*}

The NWDFT module in NWChem handles this situation by conceptually wrapping the functional routines used (the actual implementation just scales \(\tau\) by a factor 2)

\begin{eqnarray*} e &=& g(\rho,\gamma,\tau_1) \\\ g(\rho,\gamma,\tau_1) &=& f(\rho,\gamma,2\tau_1) \\\ \end{eqnarray*}

As a result, however, the density functional routines now return

\begin{eqnarray*} e &=& g(\rho,\gamma,\tau_1) \\\ Amat = &=& \frac{\partial g(\rho,\gamma,\tau_1)}{\partial\rho} \\\ Cmat = &=& \frac{\partial g(\rho,\gamma,\tau_1)}{\partial\gamma} \\\ Mmat = &=& \frac{\partial g(\rho,\gamma,\tau_1)}{\partial\tau_2} \\\ \end{eqnarray*}

where there is an inconsistency in the derivative wrt. to \(\tau\) as \(g\) is a function of \(\tau_1\) but differentiated wrt. \(\tau_2\).

In the NWXC module the differentation techniques deployed do not allow such inconsistencies and \(g\) is strictly differentiated wrt. \(\tau_1\). This results in a factor \(2\) difference that we need to compensate for.
